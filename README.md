# Tmibox

## Dependencies

* Elixir 1.12.3
* Phoenix 1.6.0
  * Surface UI (UI Component library for Phoenix)
  * Phoenix LiveView
* Postgres

## Usage

### Command line

* 라이브러리 설치

```bash
mix deps.get
```

* 데이터베이스 초기화하기

```bash
mix ecto.setup
```

* 서버 실행

```bash
mix phx.server
```

* 테스트 실행

```bash
mix test
```

### Debugging Tip

* 관리자 화면 들어가기 : `localhost:4000/admin`
* UI 컴포넌트 확인하기 : `localhost:4000/catalogue`
